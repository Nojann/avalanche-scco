Le souvenir

Dans un moment d'instrospection, tu quittes la montagne et te plonges dans ta mémoire.

Le souvenir se présente sous forme de video (2 minutes). Il est important pour l'expérience que tu visionnes attentivement le contenu.

Le son est désactivé volontairement, pour le bon déroulement de l'expérience, merci de ne pas le réactiver.
