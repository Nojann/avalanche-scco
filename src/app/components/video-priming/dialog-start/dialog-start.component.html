Le souvenir

Dans un moment d'instrospection, tu quittes la montagne et te plonges dans ta mémoire.

Le souvenir se présente sous forme de video. Il est important pour l'expérience que tu visionnes attentivement le contenu, il est préférable que le son soit activé.

Il n'est pas possible d'activer pour moi d'activer le son.
